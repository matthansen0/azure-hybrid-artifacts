# Azure Hybrid Artifacts

A collection of solutions, accelerators, and dashboards for managing hybrid and multi-cloud environments with Azure Arc.

## Solutions

| Solution | Description |
|----------|-------------|
| [Arc Catch-Up Patching](solutions/arc-catchup-patching/) | Automatically patches Arc-enabled machines that come back online after an extended offline period, bringing them into compliance with their maintenance configuration. |
